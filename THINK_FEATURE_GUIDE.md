# 🤔 Think 功能使用指南

## 功能概述

Think 功能是 Qwen3 系列模型的特色功能，允许模型在生成回答之前进行"思考"，并将思考过程展示给用户。这有助于：

1. **提高输出质量**：模型通过思考过程能够更好地分析问题
2. **增强透明度**：用户可以看到 AI 的推理过程
3. **调试和优化**：帮助理解模型的决策逻辑

## 如何使用

### 1. 模型要求
- 仅支持 **Qwen3 系列模型**（如 qwen3:4b, qwen3:8b 等）
- 其他模型将显示提示信息但不会显示开关

### 2. 启用步骤
1. 在"模型设置"页面配置 Qwen3 模型
2. 在"生成测试用例"页面的侧边栏中找到 "🤔 启用 Think 功能" 选项
3. 勾选该选项
4. 输入需求并点击"生成测试用例"

### 3. 功能效果
- **启用时**：会显示 AI 的思考过程，包括分析步骤和推理逻辑
- **禁用时**：直接显示最终结果，不显示思考过程

## 技术实现

### 控制机制
- 使用 `/think` 标签启用思考模式
- 使用 `/no_think` 标签禁用思考模式
- 支持流式输出，实时显示思考过程

### 输出格式
```
<think>
这里是 AI 的思考过程...
分析需求...
制定策略...
</think>

{
  "test_cases": [...]
}
```

## 注意事项

1. **性能影响**：启用 Think 功能可能会增加生成时间
2. **输出长度**：思考过程会增加总输出长度
3. **模型兼容性**：确保使用的是 Qwen3 系列模型
4. **网络要求**：需要稳定的网络连接以支持流式输出

## 故障排除

### 常见问题

**Q: 为什么看不到 Think 功能开关？**
A: 请确保选择的是 Qwen3 系列模型，其他模型不支持此功能。

**Q: 启用后没有显示思考过程？**
A: 检查模型配置是否正确，确保网络连接稳定。

**Q: 思考过程显示不完整？**
A: 这可能是网络问题导致的流式输出中断，可以尝试重新生成。
